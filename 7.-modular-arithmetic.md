## Modular Arithmetic: refresher.

x **is congruent** to y **modulo** m or “x ≡y (mod m)” if and only if:&#x20;

* (x−y ) is divisible by m.
* or x and y have the same remainder w.r.t. m.
* or x = y + km for some integer k.

Mod 7 equivalence classes:

{. . . ,−7, 0, 7, 14, . . .} {. . . ,−6, 1, 8, 15, . . .}...

**Useful Fact**: <mark style="color:$warning;">Addition</mark>, <mark style="color:$warning;">subtraction</mark>, <mark style="color:$warning;">multiplication</mark> can be done with any equivalent x and y.

“ a ≡c (mod m) and b ≡d (mod m) ⇒ a + b ≡c + d (mod m) and a·b= c·d (mod m)”

**Proof**:

If a ≡c (mod m), then a = c + km for some integer k.

If b ≡d (mod m), then b= d + jm for some integer j.

Therefore, a + b= c + d + (k + j)m and since k + j is integer.

⇒ a + b ≡c + d (mod m).

Can calculate with representative in {0, . . . , m−1}.

## <mark style="color:$warning;">Greatest Common Divisor</mark> and Inverses.

**Thm**:

<mark style="color:$success;">If</mark> <mark style="color:$success;"></mark><mark style="color:$success;">**greatest common divisor**</mark> <mark style="color:$success;"></mark><mark style="color:$success;">of</mark> $$x$$ <mark style="color:$success;">and</mark> $$m$$<mark style="color:$success;">,</mark> $$gcd(x, m)$$<mark style="color:$success;">, is 1, then</mark> $$x$$ <mark style="color:$success;">**has a multiplicative inverse modulo**</mark> $$m$$<mark style="color:$success;">.</mark>

**Proof**:

**Claim**:&#x20;

The set $$S= \{ 0x, 1x, . . . , (m−1)x \}$$ contains $$y \equiv 1 \space mod \space m$$ if all <mark style="color:$warning;">distinct modulo</mark> $$m$$.

{% hint style="info" %}
1, Each number in $$S$$ modulo m, which means the remainder dividing by m, is distinct from everyone else.

2, List all the numbers, what i am looking for, is that if I ever see one in the set then I found the multiplicative inverse.
{% endhint %}

Each of $$m$$ numbers in $$S$$ correspond to different one of $$m$$ <mark style="color:$warning;">equivalence classes modulo</mark> $$m$$.

⇒ One must correspond to 1 modulo $$m$$. <mark style="color:blue;">Inverse Exists</mark>!

**Proof of Claim**: (Bijection Proof)

* If not distinct, then $$\exists a, b \in \{ 0, ..., m-1 \}, a \neq b$$, where $$(ax \equiv bx (mod \space m))$$ ⇒$$(a−b)x \equiv 0 (mod \space m)$$. Or $$(a−b)x = k m$$ for some integer $$k$$.
* $$gcd (x, m) = 1$$ ⇒ Prime factorization of $$m$$ and $$x$$ <mark style="color:$warning;">do not</mark> contain common primes.

⇒ $$(a−b)$$ factorization <mark style="color:$warning;">contains all primes in</mark> $$m$$<mark style="color:$warning;">’s factorization</mark>. (<mark style="color:$danger;">WHY???</mark>)

So $$(a−b)$$ has to be multiple of $$m$$. ⇒ $$(a−b) ≥m$$.&#x20;

But $$a, b ∈{0, ...m−1}$$. <mark style="color:$danger;">Contradiction</mark>.

{% hint style="info" %}
What is prime factorization?

A number $$m$$ to be factored into multiplication of a sequence of prime numbers. 7 = 1 \* 7, 6 = 1 \* 2 \* 3.



Why is that "⇒ $$(a−b)$$ factorization <mark style="color:$warning;">contains all primes in</mark> $$m$$<mark style="color:$warning;">’s factorization</mark>."

Because $$(a−b)x = k m$$, somehow all the factors have to be over here ($$a-b$$) on the left. Because they are not in $$x$$, they must be at least in $$a-b$$, Because there is an equality. So what ever there are factors in $$m$$, the factorization is unique, there is only one way to write it down. Once you write it down, it has to contain, the left side has to contain all the factors in $$m$$, even with multiplicity. Because the factors in $$m$$ have to be somewhere on the left side, they can not be in $$x$$, so they must all be in $$a-b$$.

Then  $$(a−b)$$ has to be multiple of $$m$$, it has to contain all the factors of $$m$$.
{% endhint %}
