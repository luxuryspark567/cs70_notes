## Modular Arithmetic: Refresher.

$$x$$ **is** <mark style="color:$warning;">congruent</mark> to $$y$$ <mark style="color:$warning;">modulo</mark> $$m$$ or “$$x \equiv y (mod \space m)$$” if and only if:&#x20;

* $$(x−y)$$ is divisible by $$m$$.
* or $$x$$ and $$y$$ have the same remainder modulo $$m$$.
* or $$x = y + km$$ for some integer $$k$$.

Mod 7 equivalence classes:

{. . . ,−7, 0, 7, 14, . . .} {. . . ,−6, 1, 8, 15, . . .}...

**Useful Fact**: <mark style="color:$warning;">Addition</mark>, <mark style="color:$warning;">subtraction</mark>, <mark style="color:$warning;">multiplication</mark> can be done with any equivalent $$x$$ and $$y$$.

“ $$a ≡ c (mod \space m)$$ and $$b≡d (mod \space m)$$ ⇒ $$a + b ≡c + d (mod \space m)$$ and $$a·b= c·d (mod \space m)$$”

**Proof**:

If $$a ≡c (mod \space m)$$, then $$a = c + km$$ for some integer $$k$$.

If $$b ≡d (mod \space m)$$, then $$b= d + jm$$ for some integer $$j$$.

Therefore, $$a + b= c + d + (k + j)m$$ and since $$k + j$$ is integer.

⇒ $$a + b ≡c + d (mod \space m)$$.

## Notation

$$x (mod \space m)$$ or $$mod (x, m)$$: <mark style="color:$warning;">remainder</mark> of $$x$$ divided by $$m$$ in $$\{0, . . . , m−1\}$$.

* $$mod (x, m) = x −\lfloor \frac {x} {m} \rfloor m$$, $$\lfloor \frac {x} {m} \rfloor$$is <mark style="color:$warning;">quotient</mark>.

$$a ≡ b (mod \space m)$$.

* Says <mark style="color:$success;">two integers</mark> $$a$$ <mark style="color:$success;">and</mark> $$b$$ <mark style="color:$success;">are equivalent modulo</mark> $$m$$.
* <mark style="color:$warning;">Modulus</mark> is m.

## <mark style="color:$warning;">Greatest Common Divisor</mark> and Inverses.

**Thm**:

<mark style="color:$success;">If</mark>  $$gcd(x, m) =1$$<mark style="color:$success;">, then</mark> $$x$$ <mark style="color:$success;">has a multiplicative inverse modulo</mark> $$m$$<mark style="color:$success;">.</mark>(greatest common divisor of $$x$$ and $$m$$, $$gcd(x, m)$$)

**Proof**:

**Claim**:&#x20;

The set $$S= \{ 0x, 1x, . . . , (m−1)x \}$$ contains $$y \equiv 1 \space mod \space m$$ if <mark style="color:$warning;">all distinct</mark> <mark style="color:$warning;">modulo</mark> $$m$$.

{% hint style="info" %}
1. Each number in $$S$$ modulo m, which means the remainder dividing by m, is distinct from everyone else.
2. List all the numbers, what i am looking for, is that if I ever see "one" in the set then I found the multiplicative inverse.
3. Example:&#x20;
   1. If $$m=5$$，$$x=2$$，then $$S=\{0,2,4,1,3\}$$, all distinct, claim stands.
   2. if $$m=6$$，$$x=2$$，then $$S=\{0,2,4,0,2,4\}$$, not all distinct, claim does not stands.
{% endhint %}

Each of $$m$$ numbers in $$S$$ correspond to different one of $$m$$ <mark style="color:$warning;">**equivalence classes**</mark> <mark style="color:$warning;"></mark><mark style="color:$warning;">modulo</mark> $$m$$.

⇒ One must correspond to 1 modulo $$m$$. <mark style="color:blue;">Inverse Exists</mark>!

{% hint style="info" %}
Why "<mark style="color:$warning;">one must correspond to 1 modulo</mark> $$m$$"?

If all numbers in $$S$$ are all distinct, and <mark style="color:$warning;">any number modulo</mark> $$m$$ <mark style="color:$warning;">could have at most</mark> $$m$$ <mark style="color:$warning;">possibilities</mark> $$\{0,1,2,…,m−1\}$$, then the remainders of all numbers modulo $$m$$ cover total $$m$$ possibilities. Then one must be included in $$S$$.
{% endhint %}

**Proof of Claim**: (<mark style="color:$warning;">**Bijection**</mark> <mark style="color:$warning;"></mark><mark style="color:$warning;">Proof</mark>)

* If not distinct, then $$\exists a, b \in \{ 0, ..., m-1 \}, a \neq b$$, where $$(ax \equiv bx (mod \space m))$$ ⇒$$(a−b)x \equiv 0 (mod \space m)$$. ⇒ $$(a−b)x = k m$$ for some integer $$k$$.
* $$gcd (x, m) = 1$$ ⇒ Prime factorization of $$m$$ and $$x$$ <mark style="color:$warning;">do not</mark> contain common primes.

⇒ $$(a−b)$$ factorization <mark style="color:$warning;">contains all primes in</mark> $$m$$<mark style="color:$warning;">’s factorization</mark>. (<mark style="color:$danger;">WHY???</mark>)

So $$(a−b)$$ has to be multiple of $$m$$. ⇒ $$(a−b) ≥m$$.&#x20;

But $$a, b ∈{0, ...m−1}$$. <mark style="color:$danger;">Contradiction</mark>.

{% hint style="info" %}
1. What is bijection？

A 与B 的元素一一对应。

2. What is prime factorization?

A number $$m$$ to be factored into multiplication of a sequence of prime numbers. 7 = 1 \* 7, 6 = 1 \* 2 \* 3.

3. Why is that " $$(a−b)$$ factorization <mark style="color:$warning;">contains all primes in</mark> $$m$$<mark style="color:$warning;">’s factorization</mark>."

Because $$(a−b)x = k m$$, somehow all the factors have to be over here, ($$a-b$$) on the left. Because they are not in $$x$$ (<mark style="color:$danger;">WHY???</mark>), they must be at least in $$a-b$$, because there is an equality.&#x20;

So what ever there are factors in $$m$$, the factorization is unique, there is only one way to write it down. Once you write it down, the left side has to contain all the factors in $$m$$, even with multiplicity. Because the factors in $$m$$ have to be somewhere on the left side, they can not be in $$x$$, so they must all be in $$a-b$$.

Then  $$(a−b)$$ has to be multiple of $$m$$, it has to contain all the factors of $$m$$.

4. <mark style="color:$warning;">Why the factors can not be in</mark> $$x$$?

Because $$gcd(x, m) = 1$$.

## More divisibility

Notation: $$d|x$$ means “$$d$$ divides $$x$$” or $$x = kd$$ for some integer $$k$$.

<mark style="color:$warning;">**Lemma 1**</mark><mark style="color:$warning;">: If</mark> $$d|x$$ <mark style="color:$warning;">and</mark> $$d|y$$ <mark style="color:$warning;">then</mark> $$d|y$$ <mark style="color:$warning;">and</mark> $$d| mod (x, y )$$<mark style="color:$warning;">.</mark>

**Proof:**

$$mod(x, y)$$ = $$x−⌊x/y⌋·y$$&#x20;

\= $$x−⌊s⌋·y$$ for integer $$s$$

\= $$kd−sℓd$$ for integers $$k$$ , $$ℓ$$ where $$x = kd$$ and $$y= ℓd$$

\= $$(k−sℓ)d$$

Therefore $$d | mod (x, y)$$. And $$d|y$$ since it is in condition.

<mark style="color:$warning;">**Lemma 2:**</mark> <mark style="color:$warning;"></mark><mark style="color:$warning;">If</mark> $$d|y$$ <mark style="color:$warning;">and</mark> $$d|mod (x, y)$$ <mark style="color:$warning;">then</mark> $$d|y$$ <mark style="color:$warning;">and</mark> $$d|x$$<mark style="color:$warning;">.</mark>

**Proof**: Similar. Try this at home.&#x20;

## Euclid’s algorithm.

<mark style="color:$warning;">**GCD Mod Corollary**</mark>**:** $$gcd(x, y ) = gcd(y , mod (x, y))$$.

What’s $$gcd(7, 0)$$? 7, since 7 divides 7 and 7 divides 0

What’s $$gcd(x, 0)$$? $$x$$

```
(define (euclid x y)
    (if (= y 0)
        x
    (euclid y (mod x y)))) ***
```

**Theorem:** (euclid x y)= gcd(x, y) if x ≥y.

**Proof:** Use <mark style="color:$warning;">Strong Induction</mark>.

**Base Case:** y= 0, “x divides y and x”

⇒ “x is common divisor and clearly largest.”

**Induction Step:** $$mod (x, y) < y ≤x$$ when $$x ≥y$$

call in line (\*\*\*) meets conditions plus arguments “smaller”

and by strong induction hypothesis

computes $$gcd(y , mod (x, y))$$

which is $$gcd(x, y)$$ by **GCD Mod Corollary**.

## Runtime Proof.

```
(define (euclid x y)
    (if (= y 0)
        x
        (euclid y (mod x y))))
```

**Theorem:** (euclid x y) uses $$O(n)$$ ”divisions” where $$n = b(x) ≈log_2 x$$.

**Proof:**

**Fact:**

<mark style="color:$warning;">First arg decreases by at least factor of</mark> <mark style="color:$warning;"></mark><mark style="color:$warning;">**two**</mark> <mark style="color:$warning;"></mark><mark style="color:$warning;">in</mark> <mark style="color:$warning;"></mark><mark style="color:$warning;">**two recursive calls**</mark>.

After $$2log_2 x = O(n)$$ recursive calls, argument $$x$$ is 1 bit number.

One more recursive call to finish.

1 division per recursive call.

$$O(n)$$ divisions.

**Proof of Fact:** Recall that first argument decreases every call.

**Case 1:** $$y< x/2$$, first argument is $$y$$ ⇒ true in one recursive call;

**Case 2:** Will show “$$y≥x/2$$” ⇒ “$$mod (x, y)≤x/2$$.”

$$mod (x,y)$$ is second argument in next recursive call, and **becomes the first argument in the next one**.

* When $$y ≥x/2$$, then $$⌊\frac {x} {y}⌋= 1$$, $$mod (x, y)$$ = $$x−y⌊\frac {x} {y}⌋$$= $$x−y$$ ≤ $$x−x/2$$= $$x/2$$.
* $$mod (x, y) \leq x/2$$.

## Extended GCD

**Euclid’s Extended GCD Theorem:**

<mark style="color:$warning;">For any</mark> $$x$$<mark style="color:$warning;">,</mark> $$y$$ <mark style="color:$warning;">there are integers</mark> $$a$$<mark style="color:$warning;">,</mark> $$b$$ <mark style="color:$warning;">where</mark> $$ax + by= d$$ <mark style="color:$warning;">where</mark> $$d= gcd (x, y)$$<mark style="color:$warning;">.</mark>

“<mark style="color:$warning;">Make</mark> $$d$$ <mark style="color:$warning;">out of sum of multiples of</mark> $$x$$ <mark style="color:$warning;">and</mark> $$y$$ .”

What is multiplicative inverse of $$x$$ modulo $$m$$?

By extended GCD theorem, when $$gcd(x, m) = 1$$.

$$ax + bm= 1$$

$$ax ≡1−bm ≡1 (mod \space m)$$.

So $$a$$ is multiplicative inverse of $$x (mod \space m)$$!!

Example: For x = 12 and y= 35 , gcd(12, 35) = 1.

(3)12+ (−1)35= 1.

a = 3 and b=−1.

The multiplicative inverse of 12 (mod 35) is 3.

## Extended GCD Algorithm.

```
ext-gcd(x,y)
    if y = 0 then return(x, 1, 0)
        else
            (d, a, b) := ext-gcd(y, mod(x,y))
            return (d, b, a - floor(x/y) * b)
```
