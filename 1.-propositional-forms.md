## Propositional Forms

* Proposition is simply **a statement which is either&#x20;**<mark style="color:green;">**true**</mark>**&#x20;or&#x20;**<mark style="color:red;">**false**</mark>.
  * “3 is odd"
  * Propositions should <mark style="color:red;">**not**</mark> include fuzzy terms.
    * Arnold Schwarzenegger often eats broccoli. (What is “often?")
    * Henry VIII was unpopular. (What is “unpopular?")
* Propositions **may be joined together** to form more complex statements. \
  Let P, Q, and R be variables representing propositions. <mark style="color:red;">Statements like these,</mark> <mark style="color:red;"></mark><mark style="color:red;">**with variables**</mark><mark style="color:red;">, are called</mark> <mark style="color:red;"></mark>_<mark style="color:red;">**propositional forms**</mark>_.
  * **Conjunction**: P∧Q (“P and Q"). True only when both P and Q are true.
  * **Disjunction**: P∨Q (“P or Q"). True when at least one of P and Q is true.
  * **Negation**: ¬P (“not P"). True when P is false.

{% hint style="info" %}
Note that P ∨¬P is always true, regardless of the truth value of P. A propositional form that is <mark style="color:red;">always true</mark> regardless of the truth values of its variables is called a <mark style="color:red;">**tautology**</mark>. Conversely, a statement such as P ∧¬P, which is <mark style="color:red;">always false</mark>, is called a <mark style="color:red;">**contradiction**</mark>.
{% endhint %}

## Implication

<figure><img src=".gitbook/assets/image (2) (1) (1) (1) (1) (1).png" alt="" width="248"><figcaption></figcaption></figure>

<mark style="color:red;">The most important and subtle propositional form is an</mark> <mark style="color:red;"></mark><mark style="color:red;">**implication**</mark>: $$P \implies Q$$, “<mark style="color:green;">P implies Q</mark>", which means that $$P \implies Q$$ is T，as in row 1，3 and 4。

* P is the <mark style="color:red;">hypothesis</mark>.
* Q is the <mark style="color:red;">conclusion</mark>.
* only is False if P is True and Q is False .
* <mark style="color:red;">False implies nothing</mark>: P False means Q can be True or False
* <mark style="color:red;">Anything implies true</mark>: P can be True or False when Q is True

Here are some of the different ways of saying it:

1. <mark style="color:green;">if P, then Q</mark>;
2. <mark style="color:green;">Q if P</mark>;
3. <mark style="color:green;">P only if Q</mark>: P can only be true if Q is true（P只有当Q为T的时候，P才可能为T）. since if Q is false P must be false.
4. <mark style="color:green;">P is sufficient for Q</mark>: Proving P (P is True) allows you to conclude that Q is true.
5. <mark style="color:green;">Q is necessary for P</mark>: For P to be true it is necessary that Q is true. Or if Q is false then we know that P is false.
6. <mark style="color:green;">Q unless not P</mark>:&#x20;
   1. Q could be true, no matter P is true or false. (As row 1 and row 3 in the truth table)
   2. Q is false only when P is false. (As row 4 in the truth table)

## Contrapositive and Converse

* <mark style="color:red;">Contrapositive</mark>: $$\lnot Q \implies \lnot P$$:&#x20;
  * $$P \implies Q \equiv \lnot P \lor Q \equiv \lnot (\lnot Q) \lor \lnot P \equiv \lnot Q \implies \lnot P$$.
* <mark style="color:red;">Converse</mark>: $$Q \implies P$$

## Truth Table of Implication

Example: If chemical plant pollutes river, fish die.

* P: chemical plant pollutes river.
* Q: fish die.

* An implication $$P \implies Q$$ is false <mark style="color:red;">**only when**</mark> <mark style="color:red;"></mark><mark style="color:red;">P is true and Q is false</mark>.
* $$P \iff Q \equiv (P \implies Q) \land (Q \implies P)$$
  * “<mark style="color:green;">P if and only if Q</mark>" (abbreviated “P iff Q"). Formally, we write $$P \iff Q$$.&#x20;
  * Note that $$P \iff Q$$ is true only when <mark style="color:red;">P and Q have the same truth values (both true or both false)</mark>.
* &#x20;$$P \implies Q \equiv \lnot Q \implies \lnot P$$
  * $$P \implies Q \equiv \lnot P \lor Q \equiv \lnot (\lnot Q) \lor \lnot P \equiv \lnot Q \implies \lnot P$$.
* Use propositional formulas to describe implication: $$((P \implies Q) \land P) \implies Q$$.
* <mark style="color:red;">**False implies nothing**</mark>, P is False means Q can be True or False.
* <mark style="color:red;">**Anything implies true**</mark>, P can be True or False when Q is True.
  * This means that <mark style="color:red;">many statements that sound nonsensical in English are true.</mark> Examples are statements like: “<mark style="color:red;">If pigs can fly, then horses can read</mark>" or “If 14 is odd then 1 + 2= 18." (These are statements that we never make in everyday life, but are perfectly natural in mathematics.) When an implication is stupidly true because the hypothesis is false, we say that it is _<mark style="color:red;">**vacuously true**</mark>_.
* If  $$P \implies Q$$  and $$Q$$ are True, does <mark style="color:red;">**not**</mark> mean $$P$$ is True.
* If  $$P \implies Q$$  and $$P$$ are True, then $$Q$$ is True .

## Quantifiers

Used to bound predicates: Proposition = Quantifier + Predicate:

* For all natural numbers $$n$$, $$\sum_{i=1} ^n i = \frac {n(n+1)}{2}$$.

Propositions should <mark style="color:red;">have an underlying “universe"</mark> and the statements are then <mark style="color:red;">quantified over that universe</mark>.&#x20;

* 3 is odd. (Yes)
* x2 + 3x = 5. (No)
* For all natural numbers n, n2 + n + 41 is prime. (Yes)

To express these statements mathematically we need <mark style="color:red;">two quantifiers</mark>: &#x20;

* The universal quantifier $$\forall$$ <mark style="color:red;">(“for all”)</mark>
  * $$(\forall x \in S) (P(x))$$. means “For all $$x$$ in $$S$$, $$P(x)$$ is True .”
* the existential quantifer $$\exists$$ <mark style="color:red;">(“there exists”)</mark>
  * $$(\exists x \in S)(P(x))$$ means “There exists an $$x$$ in $$S$$ where $$P(x)$$ is true.”

{% hint style="info" %}
Note that in **a finite universe**, we can express existentially and universally quantified propositions without quantifiers, <mark style="color:red;">using disjunctions and conjunctions respectively</mark>.&#x20;

For example, if our universe $$U$$ is $$\{ 1, 2, 3, 4 \}$$, then $$\exists  x P(x)$$  is logically equivalent to $$P(1) \lor P(2) \lor P(3) \lor P(4)$$, and $$\forall x P(x)$$ is logically equivalent to $$P(1)  \land P(2)  \land P(3)  \land P(4)$$.&#x20;

However, in an infinite universe, such as the natural numbers, this is not possible.
{% endhint %}

## Quantifiers..not commutative.

* In English: “there is a natural number that is the square of every natural number”.
  * (∃y ∈N) (∀x ∈N) (y = x^2) False
* In English: “the square of every natural number is a natural number.”
  * (∀x ∈N)(∃y ∈N) (y = x^2) True

### Tests:

* (a) ∀x∀yP(x, y) ⇒ ∀y∀xP(x, y). (<mark style="color:$success;">TRUE</mark>, For all $$\forall$$ can be switched if they are adjacent)
* (b) ∀x∃yP(x, y) ⇒ ∃y∀xP(x, y).&#x20;
* (c) ∃x∀yP(x, y) ⇒ ∀y∃xP(x, y).&#x20;

#### (b) Explanation

* The premise ∀x∃y P(x, y) asserts that for every x, there exists some y (which may depend on x) such that P(x, y) holds.
* The conclusion ∃y∀x P(x, y) asserts that there exists a single y that works for all x simultaneously.

The conclusion is much stronger, and in general, the existence of individual y for each x does not imply the existence of a common y for all x.

#### (c) Explanation

The first statement says that there is an x, say x′where for every y, P(x, y) is true. Thus, one can choose x = x′for the second statement and that statement will be true again for every y.

The converse implication ∀x∃yP(x, y) ⇒ ∃y∀xP(x, y) is **not** valid (as shown in the previous question).\
So the two statements are **not** equivalent.

### Equivalent or Not

$$¬∃x ∀y P(x, y) ⇒ ¬Q(x, y)$$ and $$∀x (∃y P(x, y)) ∧ (∃y Q(x, y))$$

(b) Not equivalent.

Justification: Using De Morgan’s Law to distribute the negation on the left side yields

$$∀x ∃y P(x, y) ∧Q(x, y)$$.

<mark style="color:$warning;">But ∃ does not distribute over ∧</mark>. There could exist different values of $$y$$ such that $$P(x, y)$$ and $$Q(x, y)$$ for a given x, but not necessarily the same value.

$$∀x ∃y P(x) ⇒Q(x, y)$$  and $$∀x P(x) ⇒(∃y Q(x, y))$$

(c) Equivalent.

Justification: We can rewrite the left side as $$∀x ∃y ¬P(x) ∨Q(x, y)$$ and the right side as

$$∀x ¬P(x) ∨(∃y Q(x, y))$$ . Clearly, the two sides are the same if $$¬P(x)$$ is true. If $$¬P(x)$$ is false, then the two sides are still the same, because $$∀x ∃y False ∨Q(x, y) ≡ ∀x False ∨ (∃y Q(x, y))$$.

## XOR

<figure><img src=".gitbook/assets/image (5).png" alt="" width="234"><figcaption></figcaption></figure>



* A ⊕B = (A ∧¬B) ∨(¬A ∧B)
  * Notice that there are only two instances when A ⊕B is true: (1) when A is true and B is false, or (2) when B is true and A is false. The clause (A ∧¬B) is only true when (1) is, and the clause (¬A ∧B) is only true when (2) is.
* A ⊕B = (A ∨B) ∧(¬A ∨¬B)
  * Another way to think about XOR is that exactly one of A and B needs to be true. This also means exactly one of¬A and¬B needs to be true. The clause (A ∨B) tells us at least one of A and B needs to be true. In order to ensure that one of A or B is also false, we need the clause (¬A ∨¬B) to be satisfied as well.
* A ⊕B = (A ∨B) ∧¬(A ∧B)
  * This is the same as the previous, with De Morgan’s law applied to equate (¬A ∨¬B) to ¬(A ∧B).
* Does (A ⊕B) imply (A ∨B)? Explain briefly.
  * (A ⊕B) = ⇒(A ∧¬B) ∨(¬A ∧B) = ⇒(A ∨B). When (A ⊕B) is true, at least one of A or B is true, which makes (A ∨B) true as well.

## **Statements with Multiple Quantifiers**

<mark style="color:red;">Quantifiers do</mark> <mark style="color:red;"></mark><mark style="color:red;">**not commute**</mark>.&#x20;

* Example 1: \
  Quantifying over two universes: $$T= \{ times \, when \, I \, ride \, on \, the \, subway \}$$ and $$P = \{ people \}$$.&#x20;
  * $$(\forall t \in T )(\exist p \in P)(p \ gets\ stabbed\ at\ time\ t)$$:
    * “Every time I ride the subway in New York, somebody gets stabbed." (it could be a different person each time):&#x20;
  * $$(\exist p \in P)(\forall t \in T )(p \ gets\ stabbed\ at\ time\ t)$$
    * “There is someone, such that every time I ride the subway in New York, that someone gets stabbed." (it <mark style="color:$info;">is always the same poor dude</mark> gets stabbed, shorking.)
* Example 2:
  * $$(\forall x \in Z)(\exist y \in Z)(x < y)$$:&#x20;
    * Given an integer, I can find a larger one. (True)
  * $$(\exist y \in Z)(\forall x \in Z)(x < y)$$:&#x20;
    * There is a largest integer. (False)

## Much Ado About Negation

### De Morgan’s Laws

* $$\lnot(P \land Q) \equiv (\lnot P \lor \lnot Q)$$
* $$\lnot (P \lor Q) \equiv ( \lnot P \land \lnot Q)$$

### Analogous laws

Negating propositions involving quantifiers follows <mark style="color:$warning;">**analogous law**</mark>.

* $$\lnot (\forall x P(x)) \equiv \exists x \lnot P(x)$$
  * (For all x, P(x) is true) does not hold, which means, there exists an x where P(x) does not hold
* $$\lnot (\exists x P(x))  \equiv \forall x \lnot P(x)$$
  * (There exsists an x, P(x) is true) does not hold, means that
    * there is not x where P(x) is true.
    * for all x, P(x) does not hold.

Example:

* The universe: $$\{ 1, 2, 3, 4 \}$$
* $$P(x) : x^2 > 10$$.

$$\exists x P(x)$$ is true and $$\forall x P(x)$$ is false

$$\lnot (\forall x P(x))$$ and $$\exists x \lnot P(x)$$ are true, since $$P(1)$$ is false.

$$\forall x \lnot P(x)$$ and $$\lnot (\exists x P(x))$$ are false, since $$P(4)$$ is true.&#x20;

### Analogous laws for more complex example

to push the negation operator inside the quantifiers (the quantifiers <mark style="color:red;">**“flip” as we go**</mark>):

$$\lnot (\forall x \exists y P(x, y)) \equiv \exists x \lnot ( \exists y P(x, y)) \equiv \exists x \forall y \lnot P(x, y)$$.

Let’s look at a trickier example:

* $$R: \exist x \exist y \exist z(x \neq y \land y \neq z \land z \neq x \land P(x) \land P(y) \land P(z))$$.
  * “there are <mark style="color:$warning;">at least</mark> three distinct integers x that satisfy P(x)"
* $$T:  \exist x \exist y \exist z \forall d (P(d) \implies d = x \lor d = y \lor d= z)$$.
  * “there are <mark style="color:$warning;">at most</mark> three distinct integers x that satisfy P(x)"
  * Here is an equivalent way to do it:
  * $$\forall x \forall y \forall v \forall z ((x \neq y \land y \neq v \land v \neq x \land x \neq z \land y \neq z \land v \neq z) \implies \lnot (P(x) \land P(y) \land P(v) \land P(z)))$$
* $$R \land T$$
  * “there are exactly three distinct integers x that satisfy P(x)", This is now easy: we can just use the conjunction of the two propositions above.
